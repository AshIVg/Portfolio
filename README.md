My Professional Portfolio Website
This repository contains the source code for my personal portfolio website, showcasing my professional journey, key projects, and diverse skill set in data analysis, software engineering, and web development.

🚀 Overview
This website serves as a dynamic and interactive online resume and project showcase. It is designed with a minimalist aesthetic, focusing on clean typography, ample whitespace, and subtle animations to provide a professional and engaging user experience.

✨ Features
Dynamic Hero Section: A welcoming introduction with a bold display of my name and a continuously looping, rotating text effect highlighting my key professional roles (Data Analyst, Software Engineer, Web Developer).

Streamlined Navigation: An elegant, thin navigation bar with subtle hover effects and a sticky-on-scroll feature for seamless browsing.

Subtle Background Animations: Incorporates a gentle, non-distracting background animation (e.g., slow gradient flow, floating particles, or soft ripples) to add a modern, sophisticated touch.

Professional Journey Timeline: A dedicated section detailing my professional experience in a clear, chronological timeline format, emphasizing achievements and skill development.

Project Showcase: In-depth descriptions of key projects, including:

Urban Housing Data Integration: Data collection, cleaning, and SQL-based integration using Python.

Sales Insights Dashboard: Data analysis and visualization with SQL and Power BI.

Student Management System: Full-stack web application development with React, Node.js, MongoDB, and Express.

Stock Price Predictor: Machine learning model using Python, TensorFlow, and NumPy for time series forecasting.

E-Commerce Analytics Platform: Real-time analytics and inventory management with React, Python, PostgreSQL, and Docker.

Responsive Design: Fully optimized for viewing across all devices (desktop, tablet, mobile).

🛠️ Technologies Used
Frontend: HTML5, CSS3, JavaScript (for dynamic elements and animations)

Styling: (Potentially Tailwind CSS or custom CSS, depending on implementation)

Backend/Data Processing (for project context): Python (Pandas, NumPy, TensorFlow), Node.js, Express, SQL, MongoDB, PostgreSQL, Docker, Power BI.
